# 安裝與解除「已損毀」警告教學

由於本應用程式尚未向 Apple 支付開發者憑證年費 (Apple Developer Program)，因此當您從網路下載並解壓縮本應用程式後，macOS 的 Gatekeeper 安全機制會自動將其標記為「已損毀」或「無法打開」。

這是所有開源或免費 Mac 軟體的常見情況，請放心，這**並非病毒**。您只需要透過一行簡單的終端機指令，解除 macOS 的隔離保護 (Quarantine) 即可永久正常使用。

### 🚀 簡單 3 步驟搞定：

1. **移動到應用程式資料夾**
   請將解壓縮出來的 `嘴炮輸入法.app` 拖曳到您的「應用程式 (Applications)」資料夾。
   *(這一步很重要，確保之後捷徑不會跑掉)*

2. **打開終端機 (Terminal)**
   您可以按下 `Command + 空白鍵` 叫出 Spotlight，輸入 `Terminal` 或 `終端機` 並按下 Return (Enter) 打開它。

3. **複製貼上解除指令**
   在終端機中，完整複製並貼上以下指令，然後按下 Return (Enter)：

   ```
   xattr -dr com.apple.quarantine /Applications/嘴炮輸入法.app
   ```

   *(這個指令會刪除 macOS 對該應用程式打上的 quarantine 標記，只解除「下載自網路」的隔離旗標，不影響其他 metadata。)*

---

### ⚙️ 首次啟動會請求兩項權限，請依序授權

第一次打開 `嘴炮輸入法.app` 後，macOS 會跳出兩個權限提示：

1. **輔助使用 (Accessibility)**
   嘴炮輸入法需要監聽全域快捷鍵並把辨識結果自動貼回前景視窗。請到 **系統設定 → 隱私權與安全性 → 輔助使用**，把 `嘴炮輸入法` 打開。

2. **麥克風 (Microphone)**
   語音辨識的核心是錄音，所以一定需要麥克風權限。請到 **系統設定 → 隱私權與安全性 → 麥克風**，把 `嘴炮輸入法` 打開。

兩個權限都打開後，可能需要重啟 app 一次才會生效。

---

**🎉 大功告成！**
現在您可以從 Finder 或 Launchpad 中正常開啟 `嘴炮輸入法.app`，享受光速般的語音辨識體驗了！
不會再跳出煩人的警告視窗。
