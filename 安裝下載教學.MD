# 🚀 VoiceType4TW 安裝與模型下載教學 (Windows)

本文件旨在解決安裝過程中常見的環境建立失敗（權限問題）以及模型下載緩慢/卡住的問題。

---

## 📂 1. 模型存放路徑 (Model Directory)

VoiceType4TW 的語音識別模型 (Whisper Models) 統一存放於以下路徑：

`%APPDATA%\VoiceType4TW\whisper_models`

> **提示：** 您可以按下 `Win + R` 鍵，輸入 `%APPDATA%\VoiceType4TW` 並回車，即可看到 `whisper_models` 資料夾。

---

## ⬇️ 2. 手動下載模型指令

如果您在 App 介面中下載模型一直卡住，可以使用以下指令手動下載。請先進入上述的模型路徑目錄，然後執行對應指令。

### 推薦：使用 Git 下載 (速度較快)

請依序複製以下指令至 **CMD** 或 **PowerShell** 執行：

#### 方案 A：Medium 模型 (平衡推薦版，約 1.5GB)
```cmd
cd %APPDATA%\VoiceType4TW\whisper_models
git clone https://huggingface.co/Systran/faster-whisper-medium medium
```

#### 方案 B：Large-v3-Turbo 模型 (旗艦極速版，最強推薦，約 1.6GB)
```cmd
cd %APPDATA%\VoiceType4TW\whisper_models
git clone https://huggingface.co/deepdml/faster-whisper-large-v3-turbo-ct2 large-v3-turbo
```

#### 方案 C：Small 模型 (低配電腦版，約 500MB)
```cmd
cd %APPDATA%\VoiceType4TW\whisper_models
git clone https://huggingface.co/Systran/faster-whisper-small small
```

> **注意：** 下載完成後，資料夾內應包含 `model.bin`、`config.json` 與 `vocabulary.json` 等檔案。

## 🛠️ 3. 安裝失敗排除 (BAT 腳本報錯)

若執行 `setup_win.bat` 時卡在「建立虛擬環境 (venv)」或「安裝依賴」階段，通常與 **磁碟寫入權限** 有關。

### ❌ 常見錯誤成因：安裝在 C 槽受保護目錄
- 安裝路徑在 `C:\` 根目錄。
- 安裝路徑在 `C:\Program Files` 或 `C:\Program Files (x86)`。
- Windows 系統會限制未授權腳本在這些位置寫入大量小檔案。

### ✅ 解決方案：
1. **更換安裝路徑 (推薦)**：
   將整個 `voicetype4tw-mac` 資料夾移至 **D 槽**、**E 槽** 或其他非系統磁碟區。
   *例如：`D:\Tools\VoiceType4TW`*

2. **移動至用戶資料夾**：
   如果只有 C 槽，請放在 `C:\Users\<您的名稱>\Documents` 或 `桌面` 執行。

3. **以系統管理員身分執行**：
   對 `setup_win.bat` 按 **右鍵** -> 選擇 **「以系統管理員身分執行」**。

---

## 💡 總結與建議
- 優先安裝在 **D 槽**。
- 如果網路不穩，先用 **Git 指令** 手動下載 `medium` 或 `large-v3-turbo`。
- 若仍有問題，請查看 `%APPDATA%\VoiceType4TW\debug.log` 獲取詳細錯誤資訊。
